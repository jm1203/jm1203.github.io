# 블로그 마이그레이션 완료

Minimal Mistakes 커스텀 테마 기반으로 성공적으로 블로그를 구축했습니다. 기존 Chirpy 테마의 복잡한 CI/CD 환경을 걷어내고, GitHub Actions 공식 배포 방식을 사용하여 깔끔하게 세팅을 마쳤습니다.

## 향후 포스팅 계획

앞으로 이 공간에서는 다음과 같은 주제들을 깊이 있게 다룰 예정입니다:

- **멀티 클라우드 로그 통합 (AWS/GCP/NCP to OCSF)**
  - 여러 클라우드 벤더의 파편화된 로그를 Open Cybersecurity Schema Framework (OCSF)로 통합하는 방법론과 실무 적용기.
- **스페이스 에셋 보안 (Space Asset Security)**
  - 우주 산업 및 위성 인프라와 관련된 사이버 보안 위협 및 방어 체계 연구.
- **각종 모의해킹 및 보안 트러블슈팅 로그**
  - 시스템 엔지니어 관점에서의 인프라 디버깅, 취약점 분석 과정.

첫 글입니다! 앞으로 잘 부탁드립니다.
